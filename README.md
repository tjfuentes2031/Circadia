The Problem:

Many students today struggle with maintaining a healthy sleep schedule due to: 

• Excessive gadget use 
• Academic workload  
• Poor time management

According to the Centers for Disease Control and Prevention, teenagers need 8-10 hours of sleep, yet many do not meet this requirement. As a result, students experience: 

• Difficulty concentrating
• Low energy levels 
• Decreased academic performance

There is a need for a simple tool that helps students track and improve their sleep habits.


What will the user input? (User Input)

The user will enter:

• Time they go to sleep
• Time they wake up


What will the program process? (Program Process)

The program will:

1. Convert sleep and wake times into numerical values 
2. Compute total sleep duration 
3. Compare the result with recommended sleep hours 
4. Determine sleep quality

Uses Python concepts like: 

• Variables
• Functions
• Conditional statements (if-else)


What will the program produce? (Program Output)

The program will display:

 •  Total hours of sleep
 •  Sleep status (Enough / Not Enough)
 •  Suggestions (e.g., “Sleep earlier”) 

Example Output:

Sleep Duration: 6 hours
Status: Not Enough
Suggestion: Try sleeping 2 hours earlier


Conclusion:

The Sleep Tracker program - CIRCADIA is:

 • Practical and easy to use
 • Based on real student needs
 • Implementable using basic Python concepts

It promotes better sleep habits and healthier lifestyles.

Why This Project is Realistic

 • Uses only beginner Python
 • No need for internet or advanced tools
 • Can be expanded in the future
